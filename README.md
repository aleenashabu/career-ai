# CAREER-AI

**CAREER-AI** is a modern career guidance platform that uses AI and machine learning to help students and professionals discover and pursue the best career paths based on their skills, interests, and learning progress.

---

## 🚀 Features

- Skill assessment and self-rating
- AI-powered career path recommendations
- Interactive dashboard and progress tracking
- Real-time ML predictions
- Course and assessment management
- Clean, fast, responsive UI

---

## ⚙️ Tech Stack

- **Frontend:** React, Vite, TailwindCSS, Framer Motion
- **Backend:** Node.js (ESM), Express, MongoDB
- **ML:** Python (scikit-learn, pandas)
- **Integration:** Node.js <-> Python direct execution

---

## 📁 Project Structure

```
career-ai/
│
├── frontend/      # React app
├── backend/       # Node.js server & API
├── ml/            # Python ML scripts & models
└── assets/        # Static assets, images, config
```

---

## 🚦 Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/jerryjames2001/career-ai
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

---

## 🎯 How It Works

1. Users rate their skills and complete assessments.
2. Backend runs Python ML prediction scripts on user data.
3. Personalized career recommendations and insights are displayed on the dashboard.
